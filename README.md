# Manifesto

A Html Manfiest Generation Plugin for CakePHP 1.3

## Adding the submodule

`git submodule add git@github.com/sams/manifesto.git plugins/manifesto`

## Manking a Manifest

`cake manifest generate <Model> <Id|Slug> -lang en -device handheld`
